# 🏁 Machine Coding & Interview Mastery

Welcome to the tiered practice roadmap. This section is structured by experience level to help you scale your architectural thinking from foundation to architect level.

## 📈 Practice Tiers

### 🟢 [LEVEL 1: Foundations](./LEVEL-1-Foundations/)
*   **Target:** SDE-1 / Junior
*   **Focus:** Clean Code, SOLID, Basic State.
*   **Key Problems:** Vending Machine, Board Games.

### 🟡 [LEVEL 2: Intermediate](./LEVEL-2-Intermediate/)
*   **Target:** SDE-2 / Professional
*   **Focus:** Design Patterns, Concurrency, Thread-Safety.
*   **Key Problems:** BookMyShow, IDM, Zepto Coupons, Rate Limiter.

### 🟠 [LEVEL 3: Advanced](./LEVEL-3-Advanced/)
*   **Target:** SDE-3 / Senior
*   **Focus:** Distributed Invariants, Infrastructure modeling.
*   **Key Problems:** AWS S3, Chat System.

### 🔴 [LEVEL 4: Architect](./LEVEL-4-Architect/)
*   **Target:** SDE-4 / Founding Engineer
*   **Focus:** AI Agentic Workflows, Low-level Engines.
*   **Key Problems:** AI Agent Workflows.
